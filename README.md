# Heart to Hearts

**라이브:** [https://h2h-psi.vercel.app/](https://h2h-psi.vercel.app/)

Next.js 앱은 **`app/`** 폴더에 있습니다.

## Vercel 배포 시 필수 설정

**Root Directory**를 반드시 **`app`**으로 설정해주세요.

1. [Vercel 프로젝트 설정 → General](https://vercel.com/halos-projects-24428129/app/settings)
2. **Root Directory** 항목에서 **Edit** 클릭
3. 값에 **`app`** 입력 후 **Save**

설정하지 않으면 "No Next.js version detected" 오류가 발생합니다.
